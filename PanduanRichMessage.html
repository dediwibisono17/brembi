Cara Membuat dan Menggunakan Rich Message

Salah satu fitur yang dibanggakan dari Applikasi Line@ adalah fitur Rich Message (Pesan Mutlimedia) . Jika anda pernah menerima pesan dari Akun Official atau Line@ yang menggunakan Gambar dan gambar tersebut bisa diklik, itulah yang disebut Rich Message. Previewnya seperti ini:
Untuk Dapat membuat Rich Message (Pesan Multimedia) seperti itu, Maka anda harus sudah mengupgrade akun Line@ anda
Untuk dapat membuat Rich Message (Pesan Multimedia) seperti itu, Maka anda harus sudah menggunakan paket Monthly Plan Pro. Seilahkan kontak Brembi Store jika ingin mengupgrade paket Monthly Plan Pro di akun line@ anda.
Cara Membuat Rich Message
  1. Login ke http://admin-official.line.me/Ini adalah website untuk menggakses akun line@ anda dari PC.
    Silakan masukan User ID dan Password Akun Line yang menjadi Admin atau Member dari akun Line@ anda
  2. Setelah Login silahkan pilih akun Line@ yang ingin anda atur
  3. Lalu tampilan selanjutnya akan seperti ini, dan klik Pesan Multimedia
  4. Lalu klik tombol “buat baru”
  5. Tampilan Selanjutnya:
    Dalam 1 konten pesan Multimedia, anda bisa memasukkan satu hingga 6 Link dalam satu gambar. Seperti yang dilihat diatas, Line@ telah menyedia template yang dapat digunakan. Satu Kotak area dapat diisi dengan 1 link.
    Berikut Penjelasan Kolom-kolom diatas:
    • Judul Pesan: Text yang akan muncul ketika anda broadcast di Menu Preview. Contoh bila user Line mengaktifkan Popup Message atau Notification, ketika mereka menerima broadcast anda maka di Smartphone mereka akan Muncul text yang anda set di sini. Mirip-mirip dengan text push notifikasi. Karena itu tidak usah terlalu panjang, karena jika terlalu panjang juga akan terpotong menjadi titik-titik di bagian yang terpotong
    • Gambar Latar Belakang: Foto yang akan digunakan untuk Pesan Multimedia ini. Tips: Silahkan gunakan Aplikasi Photoshop untuk membuat gambar yang bagus. Line@ bahkan sudah menyediakan template PSD (template file photoshop) supaya gambar anda terlihat bagus sehingga pas margin yang ada. Template dapat didownload disini
    • Text tautan: adalah text yang akan muncul ketika user tidak memiliki internet yang bagus untuk me-load gambar.
    • URL: Link yang anda taruh ketika user mengklik gambar tersebut, Jika anda ingin meletakkan link website, link instagram, link social media dll. Namun trendnya adalah meletakan Link ke salah satu postingan Timeline akun line@ anda. Link tersebut dapat diperoleh dimenu Timeline, berikut cara mendapatkannya:
  6. Jika gambar dan link sudah diatur,tinggal pencet simpan
  7. Langkah Selanjutnya adalah menuju menu “Pesan”
    Lalu klik Tulis dan Pilih “Rich Message” atau Pesan Multimedia yang ingin digunakan. Anda dapat mengatur kapan pesan tersebut akan dikirim ke followers atau Adders Akun Line@ Anda dengan pengatur tanggal dan waktu-nya. Atau jika ingin kirim saat itu juga tinggal memilih “Kirim Sekarang”
